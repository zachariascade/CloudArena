## Strategy Summary

### Core Engine
- Sacrifice creatures → trigger drain effects
- Generate tokens → repeat

### Key Combos
- Pitiless Plunderer + Sac Outlet → infinite mana
- Reveillark + Karmic Guide → infinite death triggers
- Blood Artist effects → win condition

### Play Pattern
1. Build board with cheap creatures/tokens  
2. Establish sac outlet  
3. Drop payoff (Blood Artist effects)  
4. Loop sacrifices → drain table  

---

## Power Level
- Mid to high (7–8 range)
- Can combo finish or grind value

---

## Upgrade Ideas
- Add tutors for consistency
- Add faster mana for higher power tables
- Add protection if meta is removal-heavy
