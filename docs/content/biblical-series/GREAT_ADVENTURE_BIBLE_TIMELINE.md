# Great Adventure Bible Timeline — MTG Set Design Guide

## Purpose
This document tracks:
- Which biblical books belong to each timeline period
- Which books form the **core narrative**
- Which books are **support material to mine for mechanics, flavor, characters**

---

## 🧩 Core Concept

- **Narrative Books = main storyline (set backbone)**
- **All Other Books = flavor, mechanics, characters, expansions**

You are building:

- **12 sets = 12 epochs**

  - Support books = depth + card ideas

This is effectively:
> A full custom MTG multiverse grounded in a single coherent narrative arc
